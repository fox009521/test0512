前言

删除回复

相关信息 内容
访问类型 POST
访问地址 /channel/replydelete

请求参数

参数名称 是否必填 类型 说明 示例数据
userid 是 字符串 用户GID
token 是 字符串 验证码
gid 否 字符串 回复GID

请求示例


{
     "userid":"用户GID",
     "token":"token",
     "gid":"回复GID"
}

返回结果示例


    {
       "code": 1,
       "message": "删除成功",
       "result_data":null
    }
